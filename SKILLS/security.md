# Security Skill

## Mission

Protect founder trust, partner credibility, and operational resilience across the Wosool platform.

## Standards

| Focus | Standard |
|---|---|
| Inputs | Validate and sanitize all inbound data |
| Abuse prevention | Rate-limit public write surfaces |
| Access control | Implement auth and RBAC before real member onboarding |
| Secrets | Use environment-based secret management |
| Review cadence | Revisit controls every release cycle |

## Current Priority

Expand beyond validation and throttling into headers, auth, incident readiness, and environment hardening.
