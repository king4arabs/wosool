# Backend Skill

## Mission

Evolve Laravel into a secure, well-tested service layer that can support real member, admin, and ecosystem workflows.

## Standards

| Focus | Standard |
|---|---|
| Validation | Use Form Requests for incoming data |
| Contract stability | Prefer additive API changes |
| Authorization | Introduce policies and RBAC before sensitive workflows |
| Tests | Protect public and authenticated endpoints with feature tests |
| Services | Keep controllers thin as domain complexity grows |

## Current Priority

Protect the public API contract now, then implement authentication and role-based access control in the next cycle.
