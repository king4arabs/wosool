# Frontend Skill

## Mission

Build a trustworthy, bilingual-ready, performant frontend for founders in Saudi Arabia and the GCC.

## Standards

| Focus | Standard |
|---|---|
| Data access | Centralize through `src/lib/api.ts` |
| Components | Prefer reusable primitives and consistent variants |
| Localization | Design for Arabic/English and RTL/LTR from the start |
| Accessibility | Use semantic structure and clear interaction states |
| Quality | Keep lint clean and builds passing |

## Current Priority

Replace static `seed.ts` dependencies on high-value public pages with live API-backed data while preserving polished UX fallbacks.
