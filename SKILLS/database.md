# Database Skill

## Mission

Keep the data model production-ready, auditable, and compatible with Saudi-first growth and future analytics.

## Standards

| Focus | Standard |
|---|---|
| Primary database | PostgreSQL for production |
| Test path | SQLite-compatible where practical |
| Migrations | Forward-only, explicit, reversible where possible |
| Integrity | Use foreign keys and clear constraints |
| Evolution | Add indexes and reporting fields deliberately |

## Current Priority

Ensure future auth, profiles, analytics, and localization work can extend the existing schema without disruptive rewrites.
