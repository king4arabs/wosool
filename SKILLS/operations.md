# Operations Skill

## Mission

Run Wosool through a disciplined execution cadence that compounds quality and organizational memory.

## Standards

| Focus | Standard |
|---|---|
| Audit first | Start every cycle with repository review |
| Document always | Update docs, SKILLS, changelog, and version metadata |
| Validate before release | Run relevant checks and record outcomes |
| Preserve work | Improve iteratively instead of resetting context |
| Release clearly | Use semantic versions and explicit change summaries |

## Current Priority

Normalize the daily enhancement loop so repository quality improves every cycle, not only during major milestones.
