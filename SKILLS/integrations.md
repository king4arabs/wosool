# Integrations Skill

## Mission

Select and integrate third-party services in a way that supports reliability, regional fit, and long-term maintainability.

## Standards

| Focus | Standard |
|---|---|
| Vendor choice | Prefer trustworthy, well-documented providers |
| Abstraction | Keep provider-specific logic behind clear boundaries |
| Regional fit | Consider Saudi and GCC payment, email, and hosting realities |
| Security | Keep credentials and webhooks tightly managed |
| Migration readiness | Avoid unnecessary vendor lock-in |

## Current Priority

Sequence integrations behind core product readiness: auth, email, analytics, monitoring, payments, and storage.
