# DevOps Skill

## Mission

Create a reliable delivery pipeline that supports frequent improvement without sacrificing quality.

## Standards

| Focus | Standard |
|---|---|
| CI/CD | Validate frontend and backend on push and pull request |
| Environments | Maintain local, preview, staging, and production separation |
| Rollbacks | Define safe rollback paths before high-risk deployments |
| Secrets | Keep environment variables out of source control |
| Observability | Add monitoring before production launch |

## Current Priority

Keep the new CI workflow healthy, expand it as auth and deployment mature, and prepare staging-grade operational checks.
